Bank Statement Generator

Generate a bank statement from the invoices/bills payment.



Add a new button in the invoice form view for call the statement bank generator.

In the statement bank generator view the user select the name of the statement and the journal to use in the bank statement.

After press the button Generate Statement the module create a new statement with a line by each payment in done in the selected invoice.

Generate Statement from multiple invoices/bills at the same time.



For generate Banks Statements for various invoices/bills at the same time is added a new option in the tree view action button. The first step is select the invoices/bills to generate the statements. Then select the action Generate Bank Statement. By each invoice will be generated a bank statement with a line by each pay.

If any of the invoices/bills selected have not at least one pay, the system will raise a validation error.

After generate, a list view will be show with the bank statements by each selected invoice.

Finally the user must to validate and conciliate each bank statement generated.
